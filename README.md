# liberty-gui

Gui for Liberty Role Play (don't distribute!!!)

---
## Project setup
```
npm install
```
or
```
yarn
```

---
## Compiles and hot-reloads for development
```
npm run serve
```
or
```
yarn run serve
```

---
## Compiles and minifies for production
```
npm run build
```
or
```
yarn run build
```

---
## Lints and fixes files
```
npm run lint
```
or
```
yarn run lint
```
